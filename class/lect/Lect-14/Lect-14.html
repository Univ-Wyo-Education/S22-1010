What is a recursive function definition:

For a positive integer:

    n! = n * (n-1) * ... * 2 * 1

or

    f(n) = n * (n-1) * ... * 2 * 1

or

    f(n) = n * f(n-1)

or

    f(n) = { n <= 1 : 1
           { n > 1  : n * f(n-1)
